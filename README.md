
# 介绍
  本代码改进于fanuc_grinding，使代码适用于ros kinetic 版本，并且将fanuc机器人改为ur5e机器人，优化轨迹剔除无法求得逆解的轨迹点。
  源码地址
  ```
  https://github.com/KevinBollore/fanuc_grinding
  ```
# 环境
ubuntu 16.04

ros kinetic
# 使用
```
roslaunch ur_grinding_rviz_plugin ur_grinding.launch
```
